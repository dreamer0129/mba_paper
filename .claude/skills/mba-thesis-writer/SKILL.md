# MBA论文写作助手

## 概述

MBA论文写作助手专为撰写中国MBA工商管理硕士学位论文而设计。支持案例分析和实证研究两种主要类型的论文，提供从选题、文献综述、研究设计到数据分析、结论撰写的全流程支持。集成常用商业分析框架(SWOT、PESTEL、波特五力、价值链、BCG矩阵、安索夫矩阵、麦肯锡7S、平衡计分卡)，帮助进行专业的案例分析。遵循中国MBA论文的标准结构和写作规范，使用学术规范的中文表达。

## 使用时机

使用本skill当用户:
- 明确提出需要撰写或修改MBA论文
- 提到论文题目、论文大纲或具体章节(如绪论、文献综述、案例分析等)
- 要求应用商业分析框架(SWOT、PESTEL等)进行企业或案例分析
- 询问MBA论文写作的规范、结构或格式要求

## 使用前准备

在开始写作之前,首先收集以下信息:

1. **论文基本信息**:
   - 论文题目或研究主题
   - 论文类型(案例分析/实证研究)
   - 当前进度(选题阶段/已有大纲/部分章节已完成)

2. **具体需求**:
   - 需要撰写的章节或部分
   - 已有的资料和数据(案例资料、调查数据等)
   - 特定要求(字数、格式、截止时间等)

3. **可选信息**:
   - 学校或项目的特殊格式要求
   - 指导教师的具体建议
   - 参考的文献或案例

## 核心工作流程

### 第一步:理解用户需求

#### 确认论文类型和阶段

首先明确:
- 论文是案例分析类还是实证研究类?
- 目前处于哪个阶段?(选题、文献综述、数据收集、写作、修改等)
- 本次需要完成什么?(新写某章节/修改现有内容/优化全文等)

提问示例:
- "您的论文是案例分析类还是实证研究类?"
- "您现在需要撰写哪个章节?已经有哪些材料或大纲?"
- "论文题目是什么?研究的核心问题是什么?"

#### 收集必要信息

根据用户需求,收集相应信息:
- 如需撰写绪论:了解研究背景、问题、意义
- 如需文献综述:了解研究主题、已读文献
- 如需案例分析:了解案例企业、数据来源
- 如需实证分析:了解研究变量、数据情况

### 第二步:提供针对性写作支持

根据用户需要撰写的章节,参考相应的结构和规范提供支持。

#### 2.1 绪论章节

**参考资源**: `references/thesis_structure.md` 第一章部分

**写作重点**:
- 从宏观到微观逐层聚焦到研究问题
- 明确提出1-3个具体的研究问题
- 阐明理论意义和实践意义
- 说明研究方法和论文结构

**协助方式**:
1. 帮助梳理研究背景,建立逻辑递进
2. 协助提炼清晰、可研究的研究问题
3. 阐述研究的理论和实践价值
4. 构建论文整体框架图

**使用模板**: `assets/chapter_templates.md` 第一章模板

#### 2.2 文献综述与理论基础

**参考资源**: `references/thesis_structure.md` 第二章部分

**写作重点**:
- 界定核心概念(引用权威定义)
- 介绍理论基础(2-3个支撑理论)
- 按主题归类综述文献(非简单罗列)
- 指出研究缺口
- 提出研究框架或假设

**协助方式**:
1. 帮助查找和整理概念定义
2. 介绍相关理论的核心观点
3. 构建文献综述的主题分类结构
4. 识别并表述研究缺口
5. 绘制理论框架图或研究模型

**使用模板**: `assets/chapter_templates.md` 第二章模板

#### 2.3 研究设计与方法

**参考资源**:
- `references/thesis_structure.md` 第三章部分
- `references/research_methods.md` (详细方法论指导)

**案例研究类写作重点**:
- 说明案例选择标准和理由
- 详述数据收集方法(访谈、观察、文档)
- 说明数据分析方法(如扎根理论)
- 阐述研究质量保障措施

**实证研究类写作重点**:
- 定义和操作化所有变量
- 列出量表题项和来源
- 说明抽样和数据收集过程
- 说明数据分析方法(统计方法)

**协助方式**:
1. 帮助阐述案例选择的合理性
2. 设计访谈提纲(案例研究)
3. 提供量表题项参考(实证研究)
4. 规划数据分析步骤

**使用模板**: `assets/chapter_templates.md` 第三章模板

#### 2.4 案例描述与分析 / 实证分析

**参考资源**: `references/thesis_structure.md` 第四章部分

**案例研究类**:

**写作重点**:
- 描述案例背景和基本情况
- 按时间或主题详细描述案例事实
- 运用理论框架和商业分析工具分析案例
- 提炼关键发现和模式

**协助方式**:
1. 帮助构建案例描述的结构
2. **应用商业分析框架**(见下一节)
3. 分析案例数据,提炼主题和模式
4. 撰写分析性段落,解释机制和逻辑

**实证研究类**:

**写作重点**:
- 呈现样本特征和描述性统计
- 报告信效度检验结果
- 进行相关分析
- 检验研究假设(回归、SEM等)
- 呈现清晰的统计表格

**协助方式**:
1. 帮助设计统计表格结构
2. 解释统计结果
3. 撰写结果说明文字
4. 判断假设是否得到支持

**使用模板**: `assets/chapter_templates.md` 第四章模板

#### 2.5 研究发现与讨论

**参考资源**: `references/thesis_structure.md` 第五章部分

**写作重点**:
- 总结核心发现(3-5条)
- 阐述理论贡献
- 与现有文献对话(一致/不一致及原因)
- 讨论理论和实践含义

**协助方式**:
1. 提炼和总结核心发现
2. 识别和阐述理论贡献点
3. 比较本研究与现有文献的异同
4. 深化对发现的理论解释

**使用模板**: `assets/chapter_templates.md` 第五章模板

#### 2.6 结论与建议

**参考资源**: `references/thesis_structure.md` 第六章部分

**写作重点**:
- 简洁总结研究结论
- 提出具体可操作的管理建议
- 诚实指出研究局限
- 提出未来研究方向

**协助方式**:
1. 浓缩研究结论为简洁陈述
2. 将研究发现转化为管理建议
3. 识别研究局限
4. 建议有价值的未来研究方向

**使用模板**: `assets/chapter_templates.md` 第六章模板

### 第三步:应用商业分析框架

当用户需要进行案例分析时,积极建议和应用适合的商业分析框架。

**参考资源**: `references/business_frameworks.md` (包含8个框架的详细说明)

#### 框架选择指南

根据分析目的选择合适的框架:

| 分析目的 | 推荐框架 |
|---------|---------|
| 企业战略定位分析 | SWOT分析 |
| 宏观环境分析 | PESTEL分析 |
| 行业竞争分析 | 波特五力模型 |
| 企业竞争优势分析 | 价值链分析 |
| 业务组合管理 | BCG矩阵 |
| 增长战略选择 | 安索夫矩阵 |
| 组织诊断和变革 | 麦肯锡7S模型 |
| 战略绩效管理 | 平衡计分卡 |

**常用组合**:
- 外部环境分析: PESTEL + 波特五力
- 战略制定: SWOT + 安索夫矩阵
- 组织分析: 麦肯锡7S + 平衡计分卡

#### 框架应用步骤

1. **加载框架详情**: 查阅`references/business_frameworks.md`中对应框架的详细说明
2. **理解框架**: 掌握框架的核心概念、分析维度、应用步骤
3. **收集案例信息**: 向用户询问或基于已有资料收集分析所需信息
4. **进行框架分析**: 按框架的分析维度系统分析案例
5. **撰写分析内容**: 用学术语言呈现分析结果,包含数据和事实支持
6. **绘制图表**: 建议绘制相应的分析矩阵图或框架图

**示例**:
```
用户:"帮我分析XX公司的竞争优势"

Assistant行动:
1. 建议使用价值链分析框架
2. 查阅references/business_frameworks.md中价值链分析部分
3. 询问用户关于该公司主要活动和支持活动的信息
4. 按价值链框架(进货物流→生产运营→出货物流→市场营销→售后服务)
   分析各环节如何创造价值
5. 识别竞争优势来源
6. 撰写价值链分析段落,引用框架理论,结合案例事实
```

### 第四步:规范引用格式

确保所有文献引用符合GB/T 7714-2015标准。

**参考资源**: `references/citation_format.md`

**文中引用**:
- 中文文献: 张三(2020)认为…… 或 (张三,2020)
- 英文文献: Porter(1985)提出…… 或 (Porter, 1985)
- 三位以上作者: 张三等(2020) 或 Smith et al.(2020)

**参考文献列表**:
- 中文在前,英文在后
- 按作者姓氏字母顺序排列
- 格式参考`references/citation_format.md`中的详细示例

**协助方式**:
1. 检查文中引用格式是否正确
2. 帮助撰写参考文献列表条目
3. 确保文中引用与参考文献列表一致

### 第五步:优化和润色

对已撰写的内容进行优化。

#### 内容优化

**论证逻辑**:
- 检查论证是否严谨,结论是否有支撑
- 确保因果关系表述准确
- 避免过度推断

**学术规范**:
- 观点需有文献支撑
- 数据和事实需注明出处
- 使用客观、严谨的表述

**结构完整**:
- 检查章节结构是否完整
- 各部分是否呼应研究问题
- 逻辑是否流畅

#### 语言润色

**学术中文表达**:
- 使用正式的书面语
- 避免口语化、随意化
- 避免过于主观的措辞

**常见问题修正**:
- 将"我觉得"改为"本研究认为"
- 将"很多"改为具体数据或"大量研究"
- 将"非常重要"改为"具有重要意义"

## 写作原则

### 学术规范性

1. **引用规范**: 所有非原创观点、数据、图表必须注明出处
2. **客观严谨**: 使用客观的学术语气,避免主观臆断
3. **逻辑清晰**: 论证环环相扣,结论有据可依
4. **语言正式**: 使用正式的学术中文,避免口语化

### 内容完整性

1. **回应研究问题**: 所有章节内容都应围绕研究问题展开
2. **证据充分**: 每个论点都要有理论支撑或数据证据
3. **首尾呼应**: 结论要回应绪论中提出的问题和目标
4. **避免空泛**: 具体化描述,避免泛泛而谈

### MBA论文特点

1. **应用导向**: 强调实践价值和管理启示
2. **理论联系实际**: 理论分析要与管理实践结合
3. **案例丰富**: 使用具体的企业案例说明问题
4. **建议可操作**: 管理建议要具体、可行,而非空洞口号

## 重要提醒

### 写作过程中

1. **保持中文写作**: 除非文献作者、专有名词,全程使用中文
2. **循序渐进**: 先框架后细节,先粗稿后精修
3. **持续沟通**: 不清楚时及时向用户询问,不要臆测
4. **灵活调整**: 根据用户反馈随时调整写作方向和重点

### 不要做的事

1. ❌ 不要编造数据、文献或引用
2. ❌ 不要过度理论化而脱离实践
3. ❌ 不要简单堆砌文献,要有评述和整合
4. ❌ 不要忽视格式规范和学术诚信

### 质量标准

**良好的MBA论文应该**:
- ✓ 研究问题明确,具有理论和实践价值
- ✓ 文献综述全面,理论基础扎实
- ✓ 研究方法科学,数据分析规范
- ✓ 论证逻辑严密,结论有说服力
- ✓ 管理建议具体,具有可操作性
- ✓ 语言规范学术,格式符合要求

## 使用示例

### 示例1: 撰写文献综述

用户: "我的论文题目是'制造业企业数字化转型研究',请帮我撰写文献综述部分"

Assistant行动:
1. 询问更多信息:"您主要关注数字化转型的哪些方面?如驱动因素、实施路径、影响因素等?"
2. 查阅`references/thesis_structure.md`了解文献综述的写作要点
3. 使用`assets/chapter_templates.md`中的第二章模板构建框架
4. 建议文献综述的主题分类:
   - 数字化转型的概念界定
   - 数字化转型的驱动因素研究
   - 数字化转型的实施路径研究
   - 数字化转型的影响因素研究
   - 文献述评
5. 协助撰写各部分内容,整合文献,指出研究缺口

### 示例2: 应用SWOT分析

用户: "请用SWOT分析框架分析华为公司的战略定位"

Assistant行动:
1. 查阅`references/business_frameworks.md`中SWOT分析部分
2. 向用户询问:"您有关于华为公司的哪些资料?如公司年报、行业报告、访谈数据等?"
3. 基于用户提供的信息或公开资料,按SWOT四个维度进行分析
4. 撰写SWOT分析段落:
   - 先介绍SWOT理论(引用文献)
   - 再逐一分析优势、劣势、机会、威胁(用具体事实支持)
   - 最后提出战略建议(SO、WO、ST、WT战略)
5. 建议绘制SWOT矩阵图

### 示例3: 修改研究方法章节

用户: "我的研究方法章节写得太简单,请帮我完善"

Assistant行动:
1. 请用户提供现有的研究方法章节内容
2. 识别论文类型(案例研究/实证研究)
3. 查阅`references/research_methods.md`和`references/thesis_structure.md`
4. 对照标准要求,识别缺失内容:
   - 案例研究:缺少案例选择理由?数据收集描述过简?缺少质量保障?
   - 实证研究:缺少变量操作化定义?缺少量表题项?缺少抽样说明?
5. 逐项补充完善内容
6. 使用`assets/chapter_templates.md`中的模板确保结构完整

## 总结

MBA论文写作助手通过提供结构化的写作指导、专业的商业分析框架、规范的学术标准,帮助撰写高质量的MBA学位论文。始终以用户需求为导向,在用户的协助下,循序渐进地完成论文的撰写和优化。记住:好的MBA论文不仅有扎实的理论基础,更要有深刻的实践洞察和可操作的管理建议。
